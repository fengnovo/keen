Serverless

Endpoints

GET /index

Get user's repositories.

$ curl http://localhost:3000/index?owner=fengnovo

Params

  • owner: github user or organization name, alias: username

Response Type

interface Template {
  name: string
  owner: string
  fullname: string
  description: string
  updated: string
}

type Result = Template[]

This service is Hosted on Vercel.
